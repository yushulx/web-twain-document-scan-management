# Document Capture Solution Comparison

This single-page web app lets you:

- Walk through a **decision framework** to identify the best capture approach
- Compare physical scanners, camera capture, and network scanning side-by-side
- Try each approach **live** with real Dynamsoft SDKs


## What You Need

### License Key (all demos)

Get a free 30-day trial:  
https://www.dynamsoft.com/customer/license/trialLicense/?product=dcv&package=cross-platform

### Physical Scanner Demo

- A TWAIN (Windows), ICA (macOS), or SANE (Linux) scanner connected to your machine
- Scanner driver installed

### Camera Capture Demo

- A built-in or USB camera
- Browser camera permission (granted on first use)
- Chrome, Edge, or Firefox (latest version)

### REST API Demo

- The [Dynamic Web TWAIN Service](https://www.dynamsoft.com/web-twain/downloads) installed and running
- A scanner connected to the service host machine
- Service default endpoint: `http://127.0.0.1:18622`
